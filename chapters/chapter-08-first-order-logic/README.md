# Chapter 08: First-Order Logic

> **Source:** *Artificial Intelligence: A Modern Approach* (4th ed.), Chapter 8
> **Part:** Part III: Knowledge, Reasoning & Planning
> **Estimated time:** 12–14 hours
> **Prerequisites:** Chapter 07: Logical Agents

---

## Chapter Overview

Extend logical representation with objects, relations, functions, quantifiers, and equality. Study FOL syntax and semantics including models, interpretations, and variable bindings. Practice knowledge engineering: encoding domains such as kinship, circuits, and ontologies in first-order form.

---

## Learning Objectives

By the end of this chapter, you will be able to:

1. Write FOL sentences using predicates, quantifiers, and equality
2. Interpret models, domains, and variable assignments formally
3. Distinguish ∀ and ∃ quantifier scope and nesting
4. Translate natural-language statements into FOL accurately
5. Engineer a small domain knowledge base in first-order logic
6. Compare expressiveness of propositional vs. first-order logic
7. Identify common knowledge-engineering pitfalls and ambiguities

## Self-Assessment

1. Translate 'Every student has a mentor' into FOL.
2. What is the difference between a term and a sentence in FOL?
3. Give an example where propositional logic cannot compactly express a FOL fact.
4. Explain how a model assigns meaning to predicate symbols.
5. What is the scope of a quantifier and why does it matter?
6. How does equality interact with function symbols?
