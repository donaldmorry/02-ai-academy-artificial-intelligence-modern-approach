# Course 2 Resources

Supplementary materials for Artificial Intelligence: A Modern Approach.

## Cheat Sheets

- Agent formulation: PEAS, environment properties, state representation
- Search: BFS, DFS, UCS, A*, local search, MCTS
- Logic: entailment, CNF, resolution, unification, forward/backward chaining
- Probability: Bayes' rule, Bayesian networks, HMMs, MDPs, POMDPs

## Practice Domains

- Romania map search
- N-Queens local search
- Sudoku CSP
- Wumpus World logic
- Gridworld MDP/RL
- Smart-home knowledge representation

Use [projects/capstone](../projects/capstone/README.md) for the formal project rubric. Keep toy traces for every major algorithm so correctness can be inspected before scaling.
